# Level_Less-Game
This is a video game for Android , Web and Windows. Other version is under development. 

An infinite survival Unity game where players battle endless waves of enemies, face powerful bosses, and fight to survive as long as possible.

## ğŸ® Game Description

Level_Less-Game is an action-packed infinite survival game built with Unity. Players must navigate through endless waves of enemies, managing their limited lives while racking up kills. The game features dynamic boss battles that appear after reaching certain kill milestones, adding strategic depth to the survival experience.

## âœ¨ Features

### Core Gameplay
- **Player Movement & Attack**: Smooth, responsive controls for movement and combat
- **Enemy Spawning System**: Continuous enemy waves that challenge the player
- **Kill Counter**: Track your survival progress and eliminations
- **Boss Battles**: Epic boss encounters triggered after specific kill counts
- **Wave System**: Boss defeats return gameplay to normal enemy waves

### Player Systems
- **Health System**: 3 lives to survive the endless onslaught
- **Death & Restart**: Clean UI for restarting or quitting when defeated

### User Interface
- **Health Bar**: Visual representation of remaining lives
- **Kill Count Display**: Real-time tracking of enemy eliminations
- **Death Screen**: Restart and quit options upon defeat
