# 🍔 Food Zone – Browser-Based Food Serving Game

Food Zone is a casual browser-based food serving game developed using Unity 6 and deployed as a WebGL build.

This project was created as part of my game development portfolio to improve my Unity skills, understand casual game mechanics, and analyze market trends in the hyper-casual food serving genre.

---

## 🎮 Game Overview

In Food Zone, players serve food to customers within a limited time. As levels progress, the number of customers increases, gameplay becomes more intense, and new tools are introduced to increase difficulty.

The game focuses on:
- Time-based challenge
- Progressive difficulty
- Customer management mechanics
- Clean and simple UI for browser gameplay

---

## 🛠️ Built With

- Engine: Unity 6
- Platform: WebGL (Browser)
- Language: C#
- Target: Casual / Hyper-casual audience

---

## 📈 Core Features

- Level-based progression
- 120-second starting timer (increases with levels)
- Increasing number of customers per level
- New tools unlocked as difficulty scales
- Performance-optimized for browser play
- Clean UI designed for fast interaction

---

## 💡 Design Decisions

- Ad integration was initially implemented using Google Ads.
- Ads were removed in the current version to prevent UI blocking and improve gameplay experience.
- Focus was placed on smooth performance and player retention over monetization.
